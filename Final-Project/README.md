# Student Segmentation Final Project
This folder contains my final project files for ITAI 1371. 

## Project Overview
This project focuses on using unsupervised machine learning to segment students based on their academic behavior and performance. The goal is to identify patterns in how students study and perform, which can help improve educational support. 

## Problem
The main problem was to identify different types of students based on academic behavior. Since there was no labeled data, I used unsupervised learning to find patterns in the data. 

## Methods Used
This project uses K-Means clustering to group students based on their learning behavior. The results were visualized and interpreted to demonstrate how unsupervised learning can discover meaningful patterns in educational data. 

## Machine Learning Concepts
- Unsupervised learning
- Clustering
- K-Means
- Feature selection
- StandardScaler
- Elbow method
- PCA visualization
- Cluster interpretation
- Data visualization

## Results
This project showed how unsupervised machine learning can be used to group students based on similar characteristics. The result of this project was segmenting students into four groups based on their grades, absences, study time, and some other factors.
