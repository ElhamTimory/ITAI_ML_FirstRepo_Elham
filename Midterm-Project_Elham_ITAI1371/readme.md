# DeepShelf Midterm Project
This folder contains my midterm project for ITAI 1371. The project is called DeepShelf, and it focuses on building a book recommendation system using machine learning.

## Project Overview
The goal of this project is to predict user interest and recommends book based on their interest. Recommendation systems are useful because they help users find items that match their interests, such as books, movies, music, or products. 
In this project, I worked with book rating data and used machine learning to classify whether a user would like a book or not. 

## Problem
The main problem was to use available book and user data to predict user preferences.

## Methods Used
For this project, I used Python and machine learning methods to prepare the data, train a model, and evaluate the results. 

Main steps included:
- Loading and exploring the dataset
- Cleaning and preparing the data
- Selecting useful features
- Training classification models
- Evaluating model performance
- Comparing results using accuracy and other evaluation metrics

## Machine Learning Concepts
This project included the following machine learning concepts:
- Classification
- Feature engineering
- Train/test split
- Model training
- Model evaluation
- Accuracy
- Confusion matrix
- Precision and recall
- Recommendation systems

## Results
This project showed how machine learning can be used to predict whether a user may like a book. I learned that recommendation systems depend on good data preparation and useful features.

Through this project, I learned how recommendation systems use patterns in user behavior to make predictions.
